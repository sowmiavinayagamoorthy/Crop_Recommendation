# Crop Prediction Web Application

This web application predicts the best crop to grow based on various soil and weather conditions using a trained machine learning model. The app features an easy-to-use interface with a video background and provides results with crop predictions.

## Features

- **Accurate Crop Predictions**: Utilizes a trained machine learning model to predict the most suitable crop based on input parameters (Nitrogen, Phosphorus, Potassium, temperature, humidity, pH, and rainfall).
- **User-Friendly Interface**: Easy-to-use interface with a real-time video background for enhanced visual appeal.
- **Detailed Result Page**: Displays the predicted crop along with additional cultivation tips.
- **Seamless Navigation**: Smooth transition between home, prediction form, and result pages.

## Workflow

1. **Home Page**: Introduction to the app, its functionality, and main features. Button to navigate to the prediction page.
2. **Prediction Page**: Input form for required parameters. Submit button to get crop prediction.
3. **Result Page**: Displays the predicted crop. Button to return to the prediction form for new inputs.

Enjoy accurate crop predictions with a visually appealing and user-friendly interface!
